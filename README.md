# git-config

## One command setup
`export QS_GIT_REPO=https://github.com/andylytical/git-config; curl https://raw.githubusercontent.com/andylytical/quickstart/main/quickstart.sh | bash`


## Manual setup
1. git clone https://github.com/andylytical/git-config.git
1. cd git-config
1. ./setup.sh
